# steam-games-analysis
Steam Games Data Analysis - A beginner's exploration of what makes players happy

## Research Questions
What factors affect the positive review rate of Steam games?

## Data Source
Kaggle - Steam Games Dataset (122,611 games)

## Key Findings
1. Higher price = lower positive rate — players expect more from expensive games
2. Casual and Indie games have the highest positive rate (77.8%)
3. Racing and Simulation games have the lowest positive rate (73-74%)

## Tools Used
Python / Pandas / Matplotlib
